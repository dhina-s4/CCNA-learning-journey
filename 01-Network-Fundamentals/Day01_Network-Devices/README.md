# Day 1 — Network Devices (My Understanding)

## What problem does this solve?
Understanding how devices communicate in a network.

## Key Concepts Covered
- Network devices (Router, Switch, Firewall)
- Basic OSI idea
- Device roles in real-world networks

## What is a Network?
A network is a group of devices connected to share data and resources.
A computer network is a digital telecommunication network which allows nodes to share resources.

## Why Networks Exist
- Share files
- Access internet
- Communication (email, video calls)

## Main Network Devices

### 1. Switch ( Cisco Catalyst 9200 & 3650 )
- Connects devices within same network
- Uses MAC addresses
- Operates at Layer 2


### 2. Router ( Cisco ISR 1000 Series, ISR 900, ISR 4000 )
- Connects different networks
- Uses IP addresses
- Example: Home router connecting LAN to Internet


### 3. Firewall ( Cisco ASA 5500-x | Fire power 2100)
- Moniter & Controls network traffic based on configure rules
- Can be placed inside the network or outside the network
- Security device

## OSI Layer Mapping (Basic Idea)
- Router → Layer 3
- Switch → Layer 2
- Hub → Layer 1

## Real-Life Example
In a college:
- Switch connects computers in lab
- Router connects lab to internet
- Firewall protects network
