# Day 4 - Intro to the CLI

## What problem does this solve?

The Cisco IOS CLI allows engineers to configure, manage, and secure network devices.

## Topics Covered

- Console connections
- Terminal emulators (PuTTY)
- User EXEC mode
- Privileged EXEC mode
- Global configuration mode
- Running configuration
- Startup configuration
- Password security

## Commands Practiced

enable

configure terminal

hostname

enable password

enable secret

service password-encryption

show running-config

copy running-config startup-config

## Lab Objective

Configure hostnames and secure privileged EXEC access on Cisco devices.

## Key Learning

Configuration changes are made in running-config and must be saved to startup-config to survive a reboot.
