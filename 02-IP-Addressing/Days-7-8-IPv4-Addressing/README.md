
# Day 7 & Day 8 - IPv4 Addressing

## What problem does this solve?

Devices need unique logical addresses to communicate across networks.

IPv4 provides an addressing system that allows routers to deliver packets between different networks.

## Topics Covered

### Day 7

- IPv4 addresses
- Binary conversion
- Dotted decimal notation
- Network vs Host portions
- IPv4 address classes

### Day 8

- Classful addressing
- Network address
- Broadcast address
- First usable address
- Last usable address
- Maximum host calculation

## Lab Objective

Configure IPv4 addresses on router interfaces and verify end-to-end connectivity.

## Commands Used

hostname

interface gigabitEthernet

ip address

description

no shutdown

show ip interface brief

show running-config

ping

## Key Learning

Routers connect multiple networks by assigning IP addresses to their interfaces.
