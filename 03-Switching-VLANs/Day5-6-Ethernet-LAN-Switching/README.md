# Day 5 & Day 6 - Ethernet LAN Switching

## What problem does this solve?

Switches must efficiently deliver Ethernet frames inside a LAN.

To do this, they learn MAC addresses and build MAC address tables.

## Topics Covered

- Ethernet frames
- MAC addresses
- Ethernet switching
- Unknown unicast flooding
- Known unicast forwarding
- ARP
- ICMP
- ARP tables
- MAC address tables

## Lab Objective

Analyze Ethernet switching behavior using Packet Tracer simulation mode.

## Commands Used

show mac address-table

arp -a

ping

## Concepts Practiced

- Dynamic MAC learning
- ARP request and ARP reply
- ICMP Echo Request
- ICMP Echo Reply
- Unknown unicast flooding
- Clearing MAC tables

## Key Learning

Switches learn source MAC addresses and use MAC address tables to efficiently forward Ethernet frames.
